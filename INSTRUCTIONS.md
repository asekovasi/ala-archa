# Инструкции по настройке сайта Ала-Арча

## Добавление изображений

### 1. Фотография гор на главной странице

Поместите фотографию гор Ала-Арча в папку `public` с именем `ala-archa-mountains.jpg`.

**Рекомендации:**
- Формат: JPG или PNG
- Размер: минимум 1920x1080 пикселей
- Ориентация: горизонтальная (ландшафтная)

### 2. Официальный логотип Ала-Арчи

Замените файл `public/ala-archa-logo.svg` на официальный логотип парка.

**Альтернатива:** Если у вас есть логотип в формате PNG/JPG:
1. Поместите файл в папку `public` (например, `ala-archa-logo.png`)
2. В файле `app/page.tsx` найдите строку с `src="/ala-archa-logo.svg"` и замените на ваш файл

## Запуск проекта

```bash
# Установка зависимостей (если еще не установлены)
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен версии
npm start
```

Сайт будет доступен по адресу: http://localhost:3000

## Особенности сайта

- **3D-анимации**: Сайт использует CSS 3D трансформации и анимации
- **Интерактивность**: При движении мыши героическая секция реагирует с 3D эффектом
- **Адаптивность**: Сайт полностью адаптирован для мобильных устройств
- **Анимации появления**: Карточки с информацией появляются при прокрутке страницы
